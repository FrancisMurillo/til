Easy color calculations

Rico Sta. Cruz · March 3, 2015

Use stylus’s CLI.

$ npm i -g stylus
$ stylus -i

Stylus boost faster that Sass and has shorter syntax for color computations. Here’s lighten:

> #aaa + 10%
=> #b2b2b2

Hue shift:

> #ff0 - 10deg
=> #ffd500

Or if you really want a one-liner:

echo "*{a: #ff0 - 10deg}" | stylus -p
* { a: #ffd500; }
